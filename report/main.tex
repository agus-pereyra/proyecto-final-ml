\subsection{Métricas de Rendimiento}

Dado que se tienen dos criterios para tomar como \textit{ground-truth} (etiquetas del experto y del dispositivo Dreem) 
usamos el \textbf{coeficiente Cohan's Kappa} \cite{cohen1960coefficient} para cuantificar cuán de acuerdo están entre sí.
\cite{landis1977measurement}
\begin{equation}
    \kappa = \frac{p_o - p_e}{1 - p_e}
    \label{eq:cohen-kappa}
\end{equation}

Además, se usa una matriz de confusión para interpretar donde el modelo acierta en sus clasificaciones y donde falla. 

Por otra parte, se utiliza la metrica de F1-scsore Macro para analizar el balance de las clasificaciones entre las clases
($N = 6$ cantidad de etapas del sueño)
$$F1 = 2 \cdot \frac{Precision \times Recall}{Precision + Recall} = \frac{2 \cdot TP}{2 \cdot TP + FP + FN}.$$


\begin{equation}
F1_{Macro} = \frac{1}{N}\cdot \sum_{i=1}^N F1_i
    \label{eq:F1-score-Macro}
\end{equation}
